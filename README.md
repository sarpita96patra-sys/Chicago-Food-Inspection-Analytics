# Chicago Food Inspection Analytics

## Project Overview

This project analyzes food inspection data from the City of Chicago to identify inspection trends, operational risks, and food safety compliance patterns within the hospitality and food service industry.

The project was developed using Python, Pandas, and Matplotlib with real-world public data obtained through API extraction.

The analysis explores how data analytics can support:
- food safety monitoring
- operational compliance
- risk identification
- preventive decision-making
- public health awareness

## Industry Context

Recent global food safety incidents involving contamination outbreaks, hygiene failures, and operational risks have highlighted the growing importance of food safety analytics in the hospitality industry.

Food safety incidents can directly impact:
- customer trust
- public health
- business reputation
- operational efficiency
- regulatory compliance

This project demonstrates how analytics can help organizations identify recurring risk patterns and support proactive operational monitoring.

## Project Objectives

The objectives of this project are to:

- Analyze food inspection trends
- Study inspection outcomes and risk categories
- Identify recurring operational violations
- Understand compliance patterns in food establishments
- Create meaningful business-focused visualizations
- Explore the role of analytics in food safety and hospitality operations

## Methodology

The project followed the following analytical workflow:

1. Extracted real-world inspection data using API requests
2. Loaded and structured the dataset using Pandas
3. Performed data cleaning and preprocessing
4. Converted date fields and handled missing values
5. Conducted Exploratory Data Analysis (EDA)
6. Created visualizations to identify trends and risk patterns
7. Derived operational and business insights from the analysis

## Exploratory Data Analysis (EDA)

The analysis focused on:
- inspection result distribution
- risk category analysis
- inspection activity over time
- recurring operational violations
- food safety compliance trends

Visualizations were created using Matplotlib to better understand operational patterns within the dataset.

## Key Insights

- Most establishments successfully passed inspections, indicating overall compliance trends.
- High-risk facilities showed comparatively higher violation frequencies.
- Inspection activities increased over time, reflecting stronger regulatory monitoring efforts.
- Repeated violations highlighted recurring operational and hygiene-related challenges.
- Analytics can support preventive operational monitoring and risk identification within the hospitality industry.
