
# Chicago Food Inspection Analytics

## Project Overview
This project analyzes food inspection data from the City of Chicago to identify operational risks, inspection trends, and food safety compliance patterns.

## Industry Context
Recent global food safety incidents have highlighted the importance of analytics in hospitality and food safety operations.

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook
- API Data Extraction

## Data Source
City of Chicago Open Data Portal:
https://data.cityofchicago.org/

## Key Insights
- High-risk establishments showed higher violation frequencies
- Inspection activity increased over time
- Repeated violations highlighted operational gaps

## Business Impact
This project demonstrates how analytics can support:
- food safety monitoring
- compliance management
- operational decision-making
- public health awareness
